SVG Map Gen

For making fancy SVG maps with floaty user icons.

Usage

Edit the members.json to suit your fancy. Pass twitter OAuth client key and client secret as the parameters. Redirect stdout to a file to save.

Example

~$ gen_map.py hWIsyfaIASofaiDAvFisWaCst fpQ2x1WE5yMVR0alVkcGVpQUtpOHlLaVN5MDlqR25ETyjw > worldmap.svg
